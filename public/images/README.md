# Church Website Image Placeholders

This website uses placeholder images that can be replaced with your actual church photos.

## Required Images

### Hero Banners (1920x600px recommended)
- `hero-home.jpg` - Welcoming church exterior or worship scene
- `hero-about.jpg` - Community gathering or fellowship
- `hero-ministries.jpg` - People serving together
- `hero-sermons.jpg` - Bible or worship setting
- `hero-events.jpg` - Community event or celebration
- `hero-live.jpg` - Worship service in progress
- `hero-give.jpg` - Hands giving or serving

### Sermon Thumbnails (800x450px recommended)
- `sermon-1.jpg` through `sermon-6.jpg` - Sermon title slides or pastor speaking

### Team Photos (400x400px recommended)
- `pastor-john.jpg` - Senior Pastor photo
- `team-sarah.jpg` - Worship Pastor photo
- `team-david.jpg` - Youth Director photo
- `team-grace.jpg` - Children's Director photo
- `team-michael.jpg` - Missions Coordinator photo

### Ministry Images (800x600px recommended)
- `ministry-youth.jpg` - Youth group activity
- `ministry-kids.jpg` - Children's ministry
- `ministry-small-groups.jpg` - Small group meeting
- `ministry-missions.jpg` - Mission trip or outreach

## How to Replace Placeholders

1. Take or source high-quality photos
2. Optimize images for web (compress to reduce file size)
3. Name files according to the list above
4. Place in the `images/` directory
5. Refresh the website to see your images

## Temporary Placeholder Colors

Until you add real images, the CSS will display colored backgrounds:
- Hero banners use gradient overlays for readability
- Card images use neutral beige backgrounds
- All images have lazy loading enabled for performance

## Image Sources (free stock photos)

- Unsplash: https://unsplash.com/s/photos/church
- Pexels: https://www.pexels.com/search/church%20community/
- Pixabay: https://pixabay.com/images/search/church/

## Recommended Image Specs

- Format: JPEG for photos, PNG for graphics with transparency
- Resolution: At least 2x the display size for retina displays
- Compression: Use tools like TinyPNG or ImageOptim
- Max file size: 200-300KB per image after compression
